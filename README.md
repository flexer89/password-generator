## Password Generator

# Description

Password Generator has a simple, clear menu.

User has two options to pick. 
1. Generate random password based on symbols, letters and numbers both lower and uppercase. User have to input total password length.
2. User can pick how many letters, symbols and numbers will be generated. Password length is a sum of entered values.


# Example Input

```
Welcome to the Password Generator!
----------------------------------
  1 - Entirely random generator
  2 - Specified letters, symbols 
      and numbers amount
----------------------------------


Choose an option: 1
You chose option number 1.
Password length: 10
```

# Example Output

```
Your password: 55++8skF49
```
